Le Trameur

Le métier textométrique


Comme nous avons expliqué précédemment, la BaO 4 utilise une méthode qui n'est pas appropriée à la nature de nos fichiers. Rappelons aussi que nous voulions établir des relations entre les patrons extraits, qui correspondent dans notre cas aux entités nommées.

Pour ce faire, nous avons utilisé Le Trameur, qui permet d'annoter un texte et de générer un graphe à partir de l'annotation réalisée. Il existe plusieurs façons de procéder, que nous détaillons ici. Pour notre part, nous avons choisi la quatrième option.

Le fichier que Le Trameur prend en entrée est un fichier TXT, le résultat de la BaO 1.

En sortie :

  • un fichier TXT, qui contient le corpus sous forme d'une liste
  • un fichier TXT, étiqueté avec Le Trameur
  • un fichier XML exporté, qui est le graphe d'annotation généré
